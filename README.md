 LIAO OS NEXT

> 一个运行在创游世界4X引擎上的高完整度手机系统模拟项目

## 简介

LIAO OS及LIAO OS NEXT是通过创游世界4X图形化编辑器制作的一个手机系统模拟演示小程序。它还原了手机的各项UI界面动画效果以及各项功能，该系统借鉴了澎湃OS的UI界面，同时正在不停的逐渐加入自己的创意。

本人作者高中生在业余时间内个人兴趣开发且持续迭代中

该系统拥有丰富的可玩性及可自定义性，除了基础的图标布局、小组件、壁纸、图标、锁屏等，还支持调整桌面动画的动画曲线，内置多款动画曲线的预制包，可供系统体验以及学习兴趣交流。

⚠️ 需要注意的是该系统并非是一个真正的手机系统/UI，仅前端，没有真正的内核、底层调度等，该系统仅限在于创游世界平台内运行，支持安卓、苹果、鸿蒙卓易通。

## 核心特性

- 4X引擎深度适配（组件化、数组、云存档）
- 丝滑的非线性动画桌面动画及ui动画
- 丰富的功能及可玩性
- 自由个性化（拖拽图标、多壁纸分类、桌面动画曲线自定义）
- 真正的第三方应用生态（4X通用模板机制）
